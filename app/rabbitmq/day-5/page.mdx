## দুইটি আলাদা জিনিস

| Concept | কী survive করে | Go code knob |
| --- | --- | --- |
| Durable queue | queue definition broker restart-এর পরে থাকে | `QueueDeclare(... durable=true ...)` |
| Persistent message | message disk-এ persist হওয়ার চেষ্টা করে | `DeliveryMode: amqp.Persistent` |

## Survival Matrix

| Queue durable? | Message persistent? | Restart-এর পরে কী আশা করা যায় |
| --- | --- | --- |
| No | No | queue-ও হারাতে পারে, message-ও হারাতে পারে |
| Yes | No | queue থাকবে, in-flight/non-persistent message হারাতে পারে |
| No | Yes | queue না থাকলে benefit নেই |
| Yes | Yes | safest baseline |

```go
q, err := ch.QueueDeclare("billing.queue", true, false, false, false, nil)
if err != nil {
	log.Fatal(err)
}

err = ch.Publish("", q.Name, false, false, amqp.Publishing{
	DeliveryMode: amqp.Persistent,
	ContentType:  "application/json",
	Body:         []byte(`{"invoice_id":9001}`),
})
if err != nil {
	log.Fatal(err)
}
```
