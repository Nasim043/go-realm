## Core Terms

| টার্ম | সহজ ব্যাখ্যা | মনে রাখার উপায় |
| --- | --- | --- |
| Broker | RabbitMQ server যা message manage করে | “post office” |
| Producer | message publish করে | sender |
| Exchange | message কোন queue-তে যাবে তা decide করে | traffic police |
| Queue | message ধরে রাখে | inbox |
| Consumer | message নিয়ে process করে | worker |
| Binding | exchange আর queue-এর relation | routing rule |

```go
package main

import (
	"log"

	amqp "github.com/rabbitmq/amqp091-go"
)

func main() {
	conn, err := amqp.Dial("amqp://guest:guest@localhost:5672/")
	if err != nil {
		log.Fatal(err)
	}
	defer conn.Close()

	ch, err := conn.Channel()
	if err != nil {
		log.Fatal(err)
	}
	defer ch.Close()

	err = ch.Publish(
		"orders.direct",
		"order.created",
		false,
		false,
		amqp.Publishing{
			ContentType: "application/json",
			Body:        []byte(`{"order_id":101}`),
		},
	)
	if err != nil {
		log.Fatal(err)
	}
}
```
